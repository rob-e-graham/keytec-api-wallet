How to use Keytec

Store a key, attach it to a project, run.

Keytec injects your API keys when you run a command. Nothing is written to a file.

$ famtec add together ✓ saved to Keychain $ famtec run openclaw -- npm run dev ✓ keys injected as env vars no .env file written
